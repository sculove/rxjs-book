# RxJS로 만든 버스 노선 예제

간단한 어플리케이션 실습. 버스 노선 조회 조회하기

> 게시된 글의 내용을 인용하거나 도용하는 경우에는 꼭! 사전에 저에게 연락을 주세요.
> 
> 책의 3부 예제 - 버스 노선 검색입니다.
> 책의 내용 중 부족하거나 아쉬운 부분에 대해서 "이슈를 등록" 해주시면,
> 출판시 베타 리뷰어로 모시도록 하겠습니다.
> 
> 감사합니다.

### Installl
```bash
npm install
```

### How to run
```bash
npm run start
```

http://localhost:3000 으로 접속하셔서 보시면 됩니다.
